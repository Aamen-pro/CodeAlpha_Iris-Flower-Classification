## 📝 About the Project
This project is part of my internship at **CodeAlpha**. The main objective is to build a Machine Learning model capable of classifying Iris flowers into three distinct species based on their sepal and petal measurements. The Iris dataset is widely considered one of the most famous foundational datasets in the fields of Data Science and Machine Learning.

## 🎯 Key Objectives
- Perform Exploratory Data Analysis (EDA) to understand the underlying patterns in the dataset.
- Preprocess and prepare the data for training.
- Build and train machine learning classification models.
- Evaluate model performance and identify the most accurate classifier.

---

## 📊 The Dataset
The project utilizes the famous **Iris Dataset**, which contains 150 samples evenly distributed across 3 flower species:
1. **Iris Setosa**
2. **Iris Versicolor**
3. **Iris Virginica**

### 📏 Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

---

## 🛠️ Technologies & Tools Used
The project is built using **Python** alongside the most popular data science libraries:
- **Pandas**: For data manipulation and cleaning.
- **NumPy**: For numerical computations.
- **Matplotlib & Seaborn**: For creating appealing and informative data visualizations.
- **Scikit-Learn**: For building, training, and evaluating machine learning models (e.g., Logistic Regression, Decision Trees).
- **Jupyter Notebook**: As an interactive development environment.

---

## 🚀 Methodology
1. **Data Loading:** Importing the `Iris.csv` file and understanding its structure.
2. **Exploratory Data Analysis (EDA):** Plotting various graphs to visualize relationships between features and how they separate the different species.
3. **Data Preprocessing:** Separating features (X) and the target variable (Y), followed by splitting the data into training and testing sets.
4. **Model Training:** Applying machine learning algorithms to train the classification model.
5. **Evaluation:** Assessing model accuracy using metrics like Accuracy Score and Confusion Matrix.

## 💻 How to Run the Project
To run this project locally, please follow these steps:

1. Clone or download the repository to your local machine.
2. Ensure you have `Python` installed (using `Anaconda` is highly recommended).
3. Install the required dependencies using the following command:
   ```bash
   pip install pandas numpy matplotlib seaborn scikit-learn jupyter
   ```
4. Launch `Jupyter Notebook` from your terminal or command prompt.
5. Open the `Iris_Flower_Classification.ipynb` file and run the cells sequentially to see the analysis and results.
